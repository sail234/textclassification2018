XMLeXtensibleMarkupLanguage 是 由 SGMLStandardGeneralizedMarkupLanguage 和 HTMLHypertextMarkupLanguage 二者 派生 的 语言 。 XML 最 重要 的 特征 是 ： 被 标记 的 各个 数据 是 保持 其 含义 的 ， 因此 系统 间 交换 数据 的 可能性 极大 提高 。 XML 最 基本 的 概念 是 ： 结构化 标记 数据 ， 实现 了 “ 文档 结构化 ” 的 语言 规范 ， 即 采用 DTDDocumentTypeDefinition 文档 类型定义 的 语言 标准 。 　 　 XML 的 结构化 文档 技术 我们 知道 HTML 语言 中 文档 标记 Tag “ ＜ ” 和 “ ＞ ” 所 包容 的 内容 和 普通 的 文本 是 混 在 一起 的 ， HTML 文档 中 的 标记 所 表示 的 内容 是 预先 由 标准 确定 的 ， 标准 以外 的 标记 内容 不能 使用 。 但 对于 XML 来说 ， 标记 Tag 可以 根据 用户 要求 来 定义 标记 的 含义 。 如图 和 图 分别 表示 用 HTML 语言 和 XML 语言 编写 的 同一 信息内容 的 格式 。 图 HTML 格式 文档 图 XML 格式 文档 　 　 ① 用 标记 表达 文档 结构 　 　 使用 标记 表达 文档 的 任务 有 二 ： 一是 正确 地 表达 XML 文档 所 具有 的 “ 结构 ” ； 二是 正确 地 传达 按 文档 所 标记 的 数据 的 “ 含义 ” 。 XML 文档 数据 采用 树枝 和 叶 的 “ 树结构 ” 方式 表达 ， 图中 XML 文档 的 树结构 形式 如图所示 。 　 　 在 XML 文档 中 ， 把 ＜ （ 标记 名 ） ＞ 形式 的 “ 开始 标记 ” 和 ＜ （ ／ 标记 名 ） ＞ 形式 的 “ 结束 标记 ” 内所 包容 的 内容 称为 “ 元素 ” 。 除 元素 的 开始 标记 和 结束 标记 以外 的 内容 称为 “ 元素 内容 ” 。 在 元素 内容 中 ， 可以 把 其它 元素 作为 包容 子结构 来 标记 。 如图 中 的 计算机 元素 ： “ 制造厂商 ” 、 “ 识别 符 ” 、 “ 钟频 ” 、 “ 内存 ” 等 元素 都 作为 子结构 存在 。 图 XML 文档 的 树结构 　 　 从图 可知 ， 存在 着 包容 子结构 的 元素 树结构 本身 ， 能以 从父 元素 派 生出 的 子 节点 来 表示 。 “ 识别 符 ” 可 进一步 包含 着 用 “ 品种 ” 和 “ 型号 ” 元素 来 表示 的 包容 子结构 。 元素 内容 如 不是 包容 子结构 ， 则 作为 元素 内容 被 标记 的 数据 在 树结构 中即 为 下位 没有 子 节点 的 “ 结束 节点 ” 。 如图 和 图 中 “ 制造厂商 ” 的 元素 内容 标记 的 “ 联想 ” 即 为 结束 节点 。 在 XML 文档 的 开始 标记 中 也 规定 了 可 指定 几种 属性 的 要求 。 属性 本身 包括 名称 和 值 作为 一对 进行 标记 。 在 图 中 的 ＜ 计算机 ＞ 开始 标记 中 ， 类型 ＝ “ 个人 机 ” 作为 属性 定义 。 在 树结构 中 属性 与 没有 包容 子结构 的 元素 的 表示 相同 。 　 　 ② 用 DTD 定义 标记 的 含义 　 　 只 具有 树结构 的 XML 文档 ， 对 应用 系统 的 处理 要求 还是 不够 的 。 XML 虽有 任意 标记 的 文档 ， 但 对 其 确切 的 解释 仍 是 不 明确 的 。 所以 在 文档 的 发送 和 接收 之间 使用 有关 标记 来 定义 确切 的 含义 是 很 有 必要 的 ， 这种 工作 称为 定义 文档 类型 。 　 　 XML 的 文档 类型 以 标记 集合 来 定义 。 标记 集合 的 定义 采用 前述 的 一种 DTD 文件 来 实现 。 图为 图中 XML 文档 的 DTD 文档 类型定义 形式 。 图 XML 文档 所 对应 的 DTD 文档 类型定义 实例 　 　 其中 “ 计算机 ” 、 “ 制造厂商 ” 等 使用 图中 XML 文档 的 标记 来 定义 。 第 行为 元素 类型 声明 ， “ 计算机 ” 元素 用 “ 制造厂商 ” 、 “ 标识符 ” 和 “ 钟频 ” 等 多个 元素 共同 定义 。 第 行为 属性 列表 声明 ， 计算机 具有 类型 属性 ， 属性 值以 文本 数据 的 表达方式 定义 。 在 这样 的 文档 中 ， 应 定义 所有 表示 的 标记 ， 正确 表达 文档 中所含 数据 的 含义 。 这种 具备 规范 形式 的 DTD 格式 并 与 XML 文档 具有 一致性 的 文档 即 称为 有效 文档 。 　 　 ③ 名称 空间 分 多个 DTD 　 　 在 制定 的 XML 标准 中 ， 个 XML 文档 允许 采用 多个 DTD 。 其 目的 是 把 一般 确定 的 DTD 与 企业 独自 的 DTD 相结合 ， 但 含义 不能 有 异 ， 以 重新 利用 原有 资源 。 因此 在 XML ． 标准 中 ， 像 规范 参照 性 标记 那样 ， 只 确定 了 一种 含义 ， 所以 才 规定 了 特定 DTD 的 “ 名称 空间 ” 标准 。 具体做法 是 把 存储 DTD 的 集合 名称 与 标记 并存 ， 以防 混同 。 　 　 强化 WWW 的 表现 能力 　 　 目前 以 HTML 为 基础 的 浏览器 尚 不 具备 与 DTD 相结合 的 XML 文档 功能 。 XML 能够 超过 HTML 的 表现 能力 ， 文档 显示 的 方式 可 根据 用户 要求 而 改变 ， 具有 “ 变换 显示 ” 的 能力 。 　 　 ① 采用 XSL 确定 显示方式 XML 文档 和 DTD 文件 中 没有 标记 显示方式 的 信息 。 当要 在 XML 文档 元素 内容 中 表示 显示方式 时 ， 采用 XSL （ XMLStyleLanguage ） 语言 描述 的 XSL 文 体表 （ XSLStyleSheet ） 实现 。 XSL 新 的 标准 已 于 年月日 公布 了 年 版 的 草案 文本 。 　 　 在 XSL 中 定义 了 二种 方式 ： 一种 是 把 原来 XML 文档 的 树结构 变换 为 其它 树结构 的 规则 ， 从而 使 文档 的 显示 变为 用户 所 需要 的 内容 ； 另 一种 方法 是 使 变换 后 的 树结构 具有 词典 用语 的 含义 。 变换 后 的 树结构 用 HTML 形式 定义 时用 词典 指定 HTML 标记 。 　 　 使用 XSL 方法 时 ， 同一 文档 可 完成 多种 表达 变换 ， 能 实现 各种 显示 内容 。 由于 同一 文档 可 表现 不同 的 XSL 文 体表 ， 因此 在 WWW 页面 中 可 显示 由表 或 图形 图片 等 构成 的 画面 。 　 　 ② 变换 为 HTML 方式 以 适应 当前 主流 　 　 一般 有 二种 显示 XML 文档 的 方法 ： 一种 方法 是 由 XML 文档 和 XSL 文 体表 生成 显示 画面 的 布局 ， 并 装入 客户端 机器 ； 另 一种 方法 是 装入 服务器端 。 当前 的 实验 表明 ， 后者 更具 优势 。 今后 随着 XML 浏览器 的 普及 ， 前种 方法 将 增加 。 　 　 系统 中 显示 处理过程 为 ： XML 文档 和 DTD 文件 首先 按 XML 处理器 （ Purser ） 规则 以 树结构 展开 ， 对 不 符合 XML 句法 规则 的 文档 ， XML 处理器 提示 错误信息 。 因为 XML 的 句法 规则 很 严格 ， 不管 用 哪种 XML 处理器 都 要求 能 抽取 同样 的 树结构 。 　 　 XSL 处理 将 根据 按 树结构 展开 的 数据 和 XSL 文 体表 来 作成 显示 用 的 数据 。 这种 以 XSL 处理器 把 显示 用 的 数据 作成 HTML 文档 在 浏览器 中 显示 的 方法 不久 即可 见到 。 　 　 通过 在 服务器 中 生成 画面 数据 的 方式 ， 将 使用 XSL 处理器 生成 的 HTML 文档 存在 WWW 服务器 中 ， 用户 可以 用 现有 的 浏览器 进行 查阅 ， 如图所示 。 图 在 服务器端 处理 XML 文档 的 系统 组成 　 　 将来 如果 显示 XML 的 浏览器 可以 普及 ， 使用 XSL 处理器 把 基于 文档 树结构 的 XML 文档 变换 为 其它 树结构 的 XML 文档 的 浏览器 和 WWW 服务器 将 成为 发展 方向 。 目前 出台 的 IE ． 即 已 是 具有 XML 文档 显示 功能 的 浏览器 ， 但 作为 WWW 页面 标记 语言 的 HTML 要 考虑 全部 可换 为 XML 。 在 个人 WWW 页面 等 只 显示 文本 和 图像 的 场合 ， 用 HTML 标记 方法 是 简单 的 。 而 在 商务 应用 中 ， 使用 XML 文档 的 WWW 服务器 的 时代 已经 到来 。 　 　 ③ 用 XLink 来 定义 灵活 的 连接 　 　 通过 XML 进行 的 连接 可 具有 HTML 所 没有 的 功能 。 在 HTML 文档 中 可 使用 URL 唯一 资源 定位器 来 连接 。 连接 的 目的地 采用 由 HTML 标准 确定 的 目的 标记 “ 锚 ” 作为 指定 URL 的 目标 （ Target ） 。 在 浏览器 中 用 鼠标 点击 后 ， 则 作为 目标 所 指定 的 HTML 文档 内容 而 显示 在 浏览器 中 。 　 　 XML 文档 具有 的 连接功能 称为 “ XLink ” ， 它 可 实现 更 高级 的 连接 设置 ， 如 作为 目标 使用 则 可以 指定 XML 文档 元素 。 用 鼠标 点击 时 不但 目标 照样 显示 ， 而且 会 把 目标 内容 嵌入 在 原 文档 中来 显示 ， 这样 便 可以 实现 弹 出 显示 。 对弹 出 显示 的 多个 目标 名称 还 可 选择 显示 的 目标 。 XLink 对 指定 目标 的 显示 方法 由 XSL 文体 形式 来 规定 。 当前 XLink 已有 用 JavaScript 标记 的 实验性 产品 ， 但 支持 XLink 的 浏览器 还 没 正式 上市 。 在 WWW 工作组 中 ， XLink 规范 的 标准化 工作 正在 开发阶段 。 　 　 ④ 定义 调用 外部 处理器 的 API 　 　 由于 XML 文档 是 结构化 的 ， 因此 如果 使用 XML 外部 处理器 ， 也 可 正确 地 取出 所 需要 的 数据 。 但 在 使用 XML 文档 全部 应用 中 ， 用来 组成 XML 处理器 的 难点 在于 代价 过高 。 因此 对 从 应用程序 调用 XML 处理器 的 接口 做 了 规定 。 这种 API 应用 程序接口 称为 DOM 文档 对象 模型 方法 ， 在 用 DOM 方法 确定 API 的 标记 中 ， 可以 采用 对象 管理 组 OMG 规定 的 CORBA （ 公用 对象 需求 代理 体系 ） 所 确定 的 IDL 接口定义 语言 。 　 　 DOM 的 基本概念 是 ： 根据 XML 文档 中 对于 使用 的 “ 某某 名称 ” 标记 所 显示 的 值 ， 即可 决定 从 应用程序 中 进行 调用 的 规则 。 在 EC 电子商务 领域 中 ， 为了 实现 在 企业 间 自动 交换 的 Web 自动化 ， DOM 技术 的 应用 是 不可 缺少 的 。 　 　 如上所述 ， XML 是 具有 很多 优越 特征 的 语言 ， WC 工作组 正在 积极 开发 有关 XML 技术 的 标准 ， 业界 参与 EC 电子商务 的 各大 公司 都 在 等待 标准 的 出台 ， 未来 的 挑战 与 机遇 并存 。 XML 的 发展 动态 　 　 当前 Internet 已 成为 业界 不可 缺少 的 生命线 ， 信息系统 不管 处于 何种 形态 都 要 考虑 接入 Internet 。 在 Internet 领域 中 ， 主要 厂商 都 在 关注 的 新 技术 就是 XML 。 以 Microsoft 、 IBM 、 Oracle 等 为首 的 大 公司 都 在 以 XML 数据格式 为 标准 进行 开发 ， 大力 向 XML 系统 进行 投资 ， 其 特点 是 ： 　 　 （ ） 美国 企业界 领导 启动 大型项目 　 　 美国 使用 XML 的 大型 电子商务 项目 已经 启动 ， 预计 年 月 完成 。 使用 个人 微 系统 的 有关 公司 也 都 组成 了 RosettaNet 组织 ， 其 目标 是 优化 企业 间 信息 交换 的 处理 效率 ， 构筑 采用 XML 数据格式 的 巨大 供应链 。 　 　 有关 电子商务 发展 的 项目 如表 所示 。 表 XML 发展 项目 　 发展 范围 项目名称 项目 概况 参加 单位 企业 间 电子商务 ICE 信息 和 SGML 规范 简洁 ， 其 软件工具 价格 也 较 低 。 同时 适用 于 XML 系统 的 基础 技术 也 在 加速 发展 ， Microsoft 公司 在 Windows 的 下 一个 版本 ， 将 对象 和 文档 统一 处理 ， 采用 XML 工作 方式 的 技术 正在 发展 之中 。 OMG 工作组 利用 XML 的 交互 接口 正在 进行 媒体 数据交换 （ XMI ） 的 标准化 工作 。 XML 应用 　 　 当前 XML 主要 应用领域 体现 在 个 方面 ： 　 　 （ ） 企业 间 电子商务 。 在 企业 间 正在 从 原有 的 广泛 使用 的 EDI 技术 转向 Web － EDI ， 即 利用 WWW 实现 商品信息 的 交换 和 接发 定单 ， 从而 使 通信 成本 降低 ， 软件 成本 降低 。 由于 原有 的 HTML 应用 的 限制 ， 使 XML 成为 大势所趋 。 利用 XML 进行 电子 目录 等 产品 信息 的 交换 和 自动更新 ， 可以 实现 目录 的 自动 分类 处理 和 服务 ， 而且 可以 实现 多 国 语言 的 目录 服务 。 　 　 （ ） 知识 管理 。 XML 可以 对 各种 文档 和 资料 进行 真正 实用 的 知识 管理 。 使用 XML 的 文档 结构化 和 文档 含义 化 功能 ， 可以 统一 进行 多 项目 索引
