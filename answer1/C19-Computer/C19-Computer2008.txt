技术 　 　 ASP 是 一种 开放 技术 ， 可以 将 HTML 、 脚本 及 可 重用 的 ActiveXServer 组件 结合 在 一起 以 建立 高效 的 、 动态 的 、 基于 Web 的 应用程序 环境 。 利用 ASP ， 可以 增加 运行 在 服务器端 的 脚本 的 新 功能 ， 开发 出 动态 、 交互 的 高性能 Web 服务器端 应用 。 利用 ASP 开发 的 脚本 程序 ， 全部 运行 在 服务器端 。 　 　 ASP 支持 ActiveXServer 组件 的 使用 。 ActiveXServer 组件 通过 组件 对象 模型 COM 为 ASP 提供 了 可编程 的 界面 ， 其中 的 ADOActiveXDataObject 提供 了 与 数据库 相连 的 机制 。 ADO 包括 个 对象 ： 　 　 Command 对象 ： 定义 对 数据源 进行 操作 的 命令 。 　 　 Connection 对象 ： 建立 个 与 数据源 的 连接 。 　 　 Error 对象 ： 提供 属于 数据 访问 错误 的 细节 。 　 　 Field 对象 ： 表示 一般 数据类型 的 字 段 。 　 　 Parameter 对象 ： 表示 command 对象 的 参数 。 　 　 Property 对象 ： 表示 由 数据 提供 源 定义 ADO 对象 的 动态 特性 。 　 　 Recordset 对象 ： 表示 数据库 信息 显示 的 全部 结果 集 。 　 　 使用 ADO 对象 可以 建立 和 管理 与 数据库 的 连接 ， 从 数据库 服务器 提取 数据 、 执行 更新 、 删除 添加 数据 。 　 　 图示 出 了 Microsoft 的 各 数据 存取 组件 共同 工作 的 情况 。 SQLServerOracleFoxpro 图 MS 数据 存取 组件 间 的 关系 　 　 当 浏览器 请求 个 ASP 文件 时 ， Web 服务器 调用 ASP ， 读取 ASP 文件 的 内容 ， 执行 ASP 文件 中 的 脚本 命令 ， 然后 向 浏览器 发送 标准 的 HTML 页面 ， 如图所示 。 图 ASP 模型 　 　 ASP 文件 是 种 嵌入 了 可 在 服务器端 执行 的 脚本 的 HTML 文档 ， 与 ASP 兼容 的
