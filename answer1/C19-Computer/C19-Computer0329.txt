ASPApplicationofwebRetrievetechnologyPagesdisplay 　 引言 　 　 ASP （ ActiveServerpages ） 是 Microsoft 公司 于 年 月 推出 的 Web 应用 程序开发 技术 。 是 提供 一个 可以 集成 脚本语言 （ Vbscript 或 Jscript 等 ） 到 HTML 网页 并 作用 于 服务器端 的 脚本 环境 。 ASP 将 HTML ， ODBC 数据库 访问 和 其它 许多 新颖 技术 集于一身 ， 从而 使 Web 应用 系统 的 设计 更加 简明 而 直观 。 因而 日益 受到 广大 开发人员 的 关注 。 下面 就 ASP 数据库 查询 多页 显示 技术 的 具体 实现 进行 论述 。 　 数据库 的 连接 与 基本 查询 技术 　 　 我们 知道 ASP 与 数据库 的 连接 是 通过 其 内置 的 ADODB 组件 来 实现 这一 功能 的 。 即 首先 通过 服务器 上 “ 控制面板 ” 的 ODBC 设置 DSN ， 通过 DSN 指向 ODBC 数据库 。 　 　 然后 就要 使用 “ ServerCreateObject ” 建立 连接 对象 ， 并用 “ Open ” 打开 待 检索 的 数据库 ， 命令 格式 为 ： 　 　 SetConnServerCreateObjectADODBConnectionConnopen 数据库 名称 　 　 这里 的 数据库 名称 即 为 在 ODBC 设定 的 “ 数据 来源 名称 ” 。 而 ServerCreateObject 需要 进行 以下 说明 。 ASP 为 使 其 在 开发 Web 应用 方面 有 更 高 一级 的 功能 ， 特 提供 了 五个 内 建 对象 ： 　 　 Request 对象 ： 用于 从 使用者 那里 获取信息 。 　 　 Response 对象 ： 将 信息 传送 给 使用者 。 　 　 Server 对象 ： 提供 存取 Web 服务器 的 方法 和 属性 。 　 　 Session 对象 ： 在 一个 用户 的 多个 主页 之间 保留 和 使用 某些 共同 的 信息 。 　 　 Application 对象 ： 在 一个 ASP 应用 内 让 不同 使用者 共享 信息 。 　 　 Server 对象 的 一个 重要 方法 就是 CreateObject 。 　 　 至于 命令 中 的 ADODB ， 它 实际上 是 ASP 提供 的 五个 服务器 控件 当中 的 一个 ， 通常 称为 数据库 访问 控件 。 一个 服务器 控件 就 相当于 一个 对象 ， 它 提供 方法 和 属性 来 使用 服务器资源 。 　 　 要 想 调用 服务器 控件 ， 就 必须 首先 利用 Server 对象 的 CreateObject 方法 产生 服务器 控件 实例 ， 比如 上面 命令 ： 　 　 SetConnServerCreateObjectADODBConnection 　 　 除了 ADODB 服务器 控件 以外 ， 其余 四个 服务器 控件 是 ： 　 　 文件 访问 控件 ： 通过 FileSystemObject 对象 和 TextStream 对象 ， 提供 读写 服务器 文件 的 功能 。 　 　 广告 轮播 器 控件 ： 让 广告 图片 信息 轮流 显示 于 主页 上 。 　 　 浏览器 信息 控件 ： 提供 用户 所用 浏览器 的 功能 信息 给 Web 服务器 。 　 　 内容 链接 控件 ： 让 你 很 方便 地 使用 一个 文本文件 来 管理 超级链接 、 上 一页 、 下 一页 等 链接 功能 。 　 　 一 但 与 数据库 建立 了 连接 ， 就 可以 设定 SQL 命令 ， 并 使用 “ Execute 开始 执行 数据库 的 检索 ， 并 将 检索 结果 存储 到 RecordSets 对象 RS 。 命令 格式 如下 ： 　 　 SetRSConnExecuteSQL 命令 　 　 其中 SQL 命令 中 的 检索 条件 可以 由 HTML 的 Form 表单 中 ， 用户 所填 数据 RSFieldscount 。 RS 字段名 称 读取 指定 字段 的 数据 。 RSEOF 是 （ True ） 否 （ False ） 已 指到 数据库 文件 的 最后 一个 记录 。 RSMoveNext 将 指针 移到 下 一个 记录 。 RSMovePrev 将 指针 移到 前 一个 记录 。 RSMoveFirst 将 指针 移 到 第一个 记录 。 RSMoveLast 将 指针 移
